# 機能要件一覧

| id | functionName | description | relatedRequirement | priority |
| --- | --- | --- | --- | --- |
| FR-001 | マージ済みレコード取得API | /api/merged-records エンドポイントにて、merged_recordsテーブルからホスト名、IPアドレス、レコードタイプ、TTLを取得しJSON形式で返却する。 | REQ-API-01 | 高 |
| FR-002 | 他プロキシレコード登録API | /api/other-records エンドポイントにて簡易トークン認証を行い、他プロキシノードから送信されたレコードを受信し、other_recordsに登録する。 | REQ-API-02 | 高 |
| FR-003 | マージ済みレコードの表示 | CLIから実行可能で、DB内のmerged_recordsをコンソールに出力する画面機能。 | REQ-CLI-01 | 中 |
| FR-004 | 静的ホストの表示 | CLIから実行可能で、DBのstatic_hostsに登録されているホスト一覧をコンソールに出力する画面機能。 | REQ-CLI-02 | 中 |
| FR-005 | 静的ホストの追加 | CLIの入力プロンプトからホスト名とIPアドレスを受け取り、static_hostsテーブルへ新しい静的ホストを追加する画面機能。 | REQ-CLI-03 | 高 |
| FR-006 | 静的ホストの削除 | CLIから指定されたホスト名をもとに、特定の静的ホストをstatic_hostsテーブルから削除する画面機能。 | REQ-CLI-04 | 中 |
| FR-007 | インスタント実行モード | CLIから選択可能で、設定された間隔で待機・実行状態を維持する対話モードの画面機能。 | REQ-CLI-05 | 低 |
| FR-008 | 名前解決処理と自レコード登録 | static_hostsに登録されたホストを名前解決し、結果をself_recordsに登録する。OSキャッシュ等に起因する自己参照ループを防止する仕組みを持つ。 | REQ-CORE-01 | 高 |
| FR-009 | mDNSクエリ応答処理 | UDPポート5353でリスンしマルチキャストグループに参加。ネットワーク内からのmDNSクエリを受信しDB内のレコードに基づき応答を返す。 | REQ-CORE-02 | 高 |
| FR-010 | プロセス排他制御 | デーモンモードでの起動時に、ロックファイルを用いてプロセスの二重起動を防止する排他制御を行う。 | REQ-SYS-01 | 高 |
| FR-011 | 常駐処理（スケジューラ）管理 | バックグラウンドループとして、クリーンアップ、名前解決、プロキシ発見、レコード同期、マージ処理の各タスクを定期的に呼び出す常駐処理。 | REQ-BATCH-01 | 高 |
| FR-012 | レコードクリーンアップ（TTL管理） | 常駐処理内で定期実行され、各レコードのTTLを減算し、期限切れとなった不要なレコードを削除する。 | REQ-BATCH-02 | 高 |
| FR-013 | プロキシ発見 | 常駐処理内で定期実行され、設定されたネットワーク内や固定IPから他のmDNSプロキシノードを検索し登録する。 | REQ-BATCH-03 | 中 |
| FR-014 | レコード同期 | 常駐処理内で定期実行され、自身のself_recordsおよび中継対象のother_recordsを他のプロキシノードへ送信して同期する。 | REQ-BATCH-04 | 高 |
| FR-015 | マージ処理 | 常駐処理内で定期実行され、self_recordsとother_recordsのデータを統合し、最終的な応答用レコードとしてmerged_recordsを生成する。 | REQ-BATCH-05 | 高 |
| FR-016 | データベース初期化・マイグレーション | システム起動時にSQLiteデータベースを初期化し、テーブル生成およびカラム追加などのマイグレーションを実施する。 | REQ-SYS-02 | 高 |
| FR-017 | ログ出力・ローテーション | システムのログを出力し、日次等でローテーションする。古いログファイルはZIP圧縮してストレージを節約する。 | REQ-SYS-03 | 低 |
| FR-018 | Wi-Fi接続管理 | Picoプラットフォーム(MicroPython)環境での起動時に、設定情報を用いてWi-Fiネットワークへ接続する。 | REQ-SYS-04 | 低 |
