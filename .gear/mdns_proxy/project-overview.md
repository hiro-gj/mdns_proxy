# プロジェクト概要

## 概要
mDNS_Proxyは、mDNS (Multicast DNS) リクエストの処理・中継・名前解決を行うプロキシデーモンおよびCLIユーティリティです。WindowsやLinuxといった標準的なOS環境だけでなく、Raspberry Pi PicoなどのMicroPython環境でも動作するように設計されており、マルチプラットフォームで柔軟に利用できるアーキテクチャを採用しています。

本システムは、独自のバックグラウンドスケジューラを用いて、定期的なレコードのクリーンアップ、独自DNS名前解決、他のプロキシノードとのレコード同期、およびマージ処理を実行します。ネットワーク上に分散したプロキシ同士が連携し、収集したサービス情報を共有することで、複雑なネットワーク環境下でのサービス発見を支援します。

システム内部ではSQLiteデータベースを用いてレコードや静的ホスト情報を永続化しており、WALモードを活用してデータベースのパフォーマンスを最適化しています。また、シンプルなHTTP APIサーバーを内蔵するほか、対話形式のCLI機能も備えており、ホストの管理やレコード状態の確認が容易に行える設計となっています。
